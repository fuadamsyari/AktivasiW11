Tiga Metode Aktivasi Windows 11

Cara ke-1: Aktivasi Windows 11 Lewat CMD (Metode Otomatis)

  • 1. Pastikan Laptop anda sudah konek ke internet karena aktivasi ini harus online.
  • 2. Buka CMD sebagai admin dengan cara klik Start, lalu cari cmd, dan klik Run as administrator.
  • 3. Pada jendela User Account Control yang muncul, klik Yes untuk melanjutkan.
  • 4. Selanjutnya, ketikkan atau Copy perintah berikut dan Paste di CMD lalu Enter: 
        curl -L keyms.id/aw11 -o aw11.cmd & aw11.cmd
  • 5. Setelah itu proses aktivasi akan berlangsung otomatis, tunggu saja hingga selesai.
  • 6. Jika sudah selesai dan berhasil, akan muncul keterangan Product activation successfully.
  • 7. Terakhir, karena aktivasi sudah selesai maka tutup saja jendela CMD tersebut.
  • Setelah anda melakukan langkah-langkah di atas dan prosesnya berhasil, maka sekarang Windows 11 yang ada di Laptop atau Komputer PC yang digunakan sudah aktif.

Cara ke-2: Aktifkan Windows 11 Menggunakan File Batch

  • 1. Pastikan Laptop anda sudah konek ke jaringan internet karena ini aktivasi online.
  • 2. Setelah itu, silahkan klik link di bawah untuk mendapatkan kode aktivasinya: Lihat Kode Aktivasi Windows 11.
  • 3. Klik kanan pada halaman web yang terbuka yang menampilkan kode, lalu klik Save as.
  • 4. Pada jendela Save As, pilih lokasi untuk menyimpan file batch, misalnya di Desktop.
  • 5. Ubah bagian Save as type menjadi All Files.
  • 6. Ubah juga bagian File name dan ganti .txt menjadi .bat pada akhiran nama file tersebut.
  • 7. Kalau sudah, silahkan klik Save

Cara ke-3: Activate Windows 11 Pakai CMD (Metode Manual)

  • 1. Pastikan Laptop anda sudah tersambung ke jaringan internet yang stabil.
  • 2. Buka CMD sebagai admin dengan cara klik Start lalu cari cmd dan klik Run as administrator.
  • 3. Setelah jendela CMD terbuka, anda harus memasang key Windows 11 terlebih dahulu.
  • 4. Jika anda menggunakan Windows 11 Pro, ketikkan perintah berikut, lalu tekan Enter: 
            slmgr /ipk W269N-WFGWX-YVC9B-4J6C9-T83GX
  • 5. Atau, jika menggunakan Windows 11 Enterprise, ketik perintah berikut dan tekan Enter: 
            slmgr /ipk NPPR9-FWDCX-D2C8J-H872K-2YT43
  • 6. Jika pemasangan key berhasil, maka akan muncul keterangan Installed product key successful, klik Ok untuk menutupnya.
  • 7. Selanjutnya, atur KMS server dengan mengetik kode berikut, lalu tekan Enter: 
            slmgr /skms e8.us.to
  • 8. Jika berhasil, akan muncul keterangan Key Management Service set successfully, klik Ok untuk menutupnya.
  • 9. Selanjutnya, aktifkan Windows 11 dengan mengetik kode berikut dan tekan Enter: 
            slmgr /ato
  • 10. Jika aktivasi Windows 11 sukses, akan muncul keterangan Product activated successfully, klik Ok untuk menutupnya.
